# Земляничная поляна

## Описание
Лендинг-страница для коттеджного городка «Земляничная поляна» в Боровом (Казахстан). Цель — продажа коттеджей и домов с ценами от 52 до 99 млн тенге.

## Возможности
- Hero-секция с анимированным фоном, плавающими частицами и эффектом зума
- Анимированные счётчики (48 домов, 15 га, 800 м до озера, 2025)
- Карточки 4 объектов: коттедж, дом, дом премиум, вилла — с характеристиками и ценами
- Секция преимуществ (лес, озеро, охрана, коммуникации и др.)
- Секция «Как добраться» с маршрутами из Астаны и Кокшетау
- Секция «Как купить дом» (4 шага)
- Форма заявки с выбором объекта
- Модальное окно с заявкой на конкретный объект
- Плавающая кнопка WhatsApp
- Toast-уведомления об успешной отправке
- Sticky-навбар с мобильным бургер-меню
- Reveal-анимации при скролле
- Полностью адаптивная вёрстка (mobile-first)
